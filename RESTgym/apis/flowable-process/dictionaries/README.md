## Regeneration Process

To regenerate with fresh data from a running API:

```bash
# 1. Start Flowable container
docker run -d -p 9090:9090 -e API=flowable-process -e TOOL=manual -e RUN=1 \
  --name flowable restgym/flowable-process

# 2. Extract process definitions
curl -u admin:test http://localhost:9090/flowable-rest/service/repository/process-definitions

# 3. Start test process to get runtime IDs
curl -u admin:test -X POST -H "Content-Type: application/json" \
  -d '{"processDefinitionKey":"oneTaskProcess"}' \
  http://localhost:9090/flowable-rest/service/runtime/process-instances

# 4. Extract tasks, executions, deployments
curl -u admin:test http://localhost:9090/flowable-rest/service/runtime/tasks
curl -u admin:test http://localhost:9090/flowable-rest/service/runtime/executions
curl -u admin:test http://localhost:9090/flowable-rest/idm-api/users
curl -u admin:test http://localhost:9090/flowable-rest/idm-api/privileges

# 5. Update dictionary JSON with real IDs
```
