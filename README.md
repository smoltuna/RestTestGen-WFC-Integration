# Tesi-RESTAPI

Testing automatico API REST.

## Contenuti

- `RESTgym/` - Infrastructure per testing API REST
- `OAS/` - OpenAPI specifications
- `APISTIC-public/` - Analisi metriche OpenAPI
- `apis/` - API source repositories
- `temp/` - File temporanei build

## Quick Start

```bash
cd RESTgym
python3 build.py
python3 run.py
```

Vedi `RESTgym/README.md` per dettagli.
